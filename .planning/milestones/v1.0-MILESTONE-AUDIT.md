**skills/README.md directory structure example is inaccurate:**

The repair-loop skill author discovered this and correctly placed the script at `/etc/nixos/zeroclaw/bin/repair-loop.sh`. Future skill authors following the README example will hit the same audit rejection.

**Impact:** Correctness of existing skills is unaffected. Developer experience gap only.
