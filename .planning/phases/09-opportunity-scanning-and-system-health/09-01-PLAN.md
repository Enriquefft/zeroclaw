Output list of new leads with scores.
  4. Insert all new leads into state.db job_applications table. For each lead run sqlite3 ~/.zeroclaw/workspace/state.db with an INSERT INTO job_applications (title, company, url, status, found_date, match_score, source_platform, last_updated) VALUES. If any leads scored 70 or above, compose a WhatsApp digest listing title, company, score, and URL for each qualifying lead. Leads scoring 90 or above get an appended note recommending cold outreach per LORE.md application approach. Send digest to +51926689401 via kapso-whatsapp-cli send --to +51926689401 --text followed by the message. If no qualifying leads, exit silently without sending.
